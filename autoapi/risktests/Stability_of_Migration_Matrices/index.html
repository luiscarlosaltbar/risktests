risktests.Stability_of_Migration_Matrices.migration_matrix_stability(df, initial_ratings_col, final_ratings_col)[source]

z-tests to verify stability of transition matrices

Parameters

  • df (array-like, at least 2D) – data

  • initial_ratings_col (string) – name of column with initial ratings values

  • final_ratings_col (string) – name of column with final ratings values

Returns

  • z_df (array-like) – z statistic for each ratings pair

  • phi_df (array-like) – p-values for each ratings pair

Notes

The Null hypothesis is that p_ij >= p_ij-1 or p_ij-1 >= p_ij depending on whether the (ij) entry is below or above main diagonal

Examples

>>res = migration_matrix_stability(df=df, initial_ratings_col=’ratings’, final_ratings_col=’ratings2’) >>print(res)
